# VS Code Extension - Snippets Helper

Code snippets hữu ích cho lập trình với OpenCV.

## Phiên Bản
- v0.0.2

## Cài Đặt
1. Tải file `.vsix`
2. Mở VS Code → Extensions (Ctrl+Shift+X)
3. Click "Install from VSIX..." và chọn file
4. Restart VS Code

## Nội Dung
Bộ snippets giúp tăng tốc độ viết code khi làm việc với OpenCV và các thư viện xử lý ảnh khác.
